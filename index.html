Evolution

The Journey of the Periodic Table

The modern periodic table wasn't created in a single day. It is the result of more than a century of scientific discoveries, improvements, and brilliant ideas.
1789

Antoine Lavoisier

The beginning of classification. Lavoisier grouped 33 known elements into metals and non-metals. This was the first organized attempt to classify elements.

💡 Why it mattered Scientists finally had a logical way to organize elements instead of remembering them individually.
1829

Döbereiner's Triads

Groups of three elements showed similar properties. The middle element usually had an atomic mass close to the average of the other two.

Example: Chlorine • Bromine • Iodine
1864

Newlands' Law of Octaves

Every eighth element displayed similar properties when elements were arranged according to increasing atomic mass.

🎵 He compared the pattern to musical octaves.
1869

Mendeleev's Periodic Table

Mendeleev arranged elements by atomic mass while keeping similar properties in the same column. He even left empty spaces for undiscovered elements.

⭐ His predictions later proved amazingly accurate.
1913

Henry Moseley

Moseley discovered that atomic number—not atomic mass—is the true basis of the periodic table. This solved several problems in Mendeleev's arrangement.

This idea became the foundation of the modern periodic table.
Today

Modern Periodic Table

Scientists have identified 118 elements. The table now contains 18 groups, 7 periods and continues to expand as new elements are synthesized.

🚀 Chemistry is still evolving.

Understanding the Table

Groups and Periods

Groups

The vertical columns of the periodic table are called Groups.

  • There are 18 Groups.
  • Elements in the same group have similar chemical properties.
  • They have the same number of valence electrons.
  • Reactivity changes as we move down a group.
💡 Memory Tip: Think of Groups as Vertical Families.

Periods

The horizontal rows of the periodic table are called Periods.

  • There are 7 Periods.
  • All elements in a period have the same number of electron shells.
  • Properties change gradually from left to right.
  • Atomic number increases across a period.
💡 Memory Tip: Periods are Horizontal Rows.

Electronic Configuration

Electron Arrangement in Atoms

Electrons are arranged in different energy levels called shells. The electronic configuration of an atom helps us determine its group, period and chemical properties.

📖 What is Electronic Configuration?

Electronic configuration is the arrangement of electrons in the different shells (K, L, M, N...) around the nucleus of an atom.

💡 Electrons always fill the inner shells before moving to outer shells.

⚛ Maximum Electrons in Each Shell

Shell Name Maximum Electrons
1 K 2
2 L 8
3 M 18
4 N 32

Worked Example

Sodium (Na)

Atomic Number = 11

Electronic Configuration = 2, 8, 1

  • Number of Shells = 3 → Period 3
  • Valence Electrons = 1 → Group 1
📝 Exam Shortcut Group = Valence Electrons

Period = Number of Occupied Shells

Quick Revision Rules

  • ✅ Period = Number of occupied shells
  • ✅ Valence Electrons = Electrons in the last shell
  • ✅ Group 1–2 → Valence electrons = Group number
  • ✅ Group 13–18 → Group = Valence electrons + 10 (for main-group elements)
  • ⚠️ Helium is an exception. It has 2 valence electrons but belongs to Group 18.

Core Concept

Electronic Configuration: The Basis of the Periodic Table

The arrangement of electrons is the foundation of the modern periodic table. It determines an element's position, properties, and chemical behaviour.

⚛ Why is it Important?

The electronic configuration tells us

  • Which period an element belongs to.
  • Which group it belongs to.
  • Its valence electrons.
  • Its chemical properties.

👨‍👩‍👧 Why Elements in the Same Group are Similar

Elements in the same group have the same number of valence electrons. Therefore they usually behave in similar ways during chemical reactions.

Example: Group 1 Elements

Element Electronic Configuration Valence Electrons
Lithium (Li) 2,1 1
Sodium (Na) 2,8,1 1
Potassium (K) 2,8,8,1 1

🧪 Sodium

Sodium has one electron in its outermost shell. It easily loses this electron to form

Na → Na⁺ + e⁻

🧪 Chlorine

Chlorine has seven electrons in its outer shell. It gains one electron to form

Cl + e⁻ → Cl⁻

💡 Remember

Same outer-shell electrons → Same Group → Similar Chemical Properties

Special Cases

Exceptions in the Periodic Table

Most elements follow the rules of the periodic table, but a few important exceptions exist because their chemical properties are different from what their electronic configuration alone suggests.
⚛

Hydrogen (H)

Hydrogen has only one valence electron, so it is placed in Group 1.

  • It is a non-metal.
  • Like alkali metals, it can lose one electron.
  • Like halogens, it can also gain one electron.
  • Its properties resemble both groups, but it is placed in Group 1 because most of its characteristics are closer to the alkali metals.
🎈

Helium (He)

Helium has the electronic configuration 1s².

  • Its outer shell is completely filled.
  • According to configuration alone, it might seem to belong to Group 2.
  • However, it behaves like the noble gases.
  • Therefore, it is placed in Group 18.
🧪

Lanthanides & Actinides

These elements actually belong to Periods 6 and 7.

  • They are called the inner transition elements.
  • If placed inside the main table, it would become too wide.
  • Therefore, they are displayed separately below the periodic table.

Remember These Exceptions

Element Expected Position Actual Position Reason
Hydrogen Can resemble Groups 1 & 17 Group 1 Mostly behaves like alkali metals
Helium Group 2 Group 18 Chemically inert like noble gases
Lanthanides & Actinides Main Table Below the Table To keep the table compact and readable

Periodic Properties

Atomic Radius

📖 Definition

Atomic radius is the distance from the nucleus to the outermost electron shell. It gives an idea of the size of an atom.

💡 Simple Explanation

Imagine an atom as a house. More electron shells mean a bigger house. Fewer shells mean a smaller house.

Trend in the Periodic Table

➡ Across a Period

Atomic radius decreases from left to right.

⬅ Smaller

⬇ Down a Group

Atomic radius increases from top to bottom.

⬇ Larger

Why Does This Happen?

  • Across a period, the number of protons increases.
  • The stronger positive charge pulls electrons closer to the nucleus.
  • Therefore, the atom becomes smaller.

  • Down a group, new electron shells are added.
  • More shells make the atom larger.
🧠 Memory Trick

Across ➜ Attraction increases ➜ Radius decreases

Down ➜ More shells ➜ Radius increases

Exam Tip

If the exam asks which atom is larger, always compare number of shells first.

Periodic Properties

Ionization Energy

📖 Definition

Ionization energy is the minimum amount of energy required to remove the outermost electron from an isolated gaseous atom.

💡 Easy Explanation

Think of the nucleus as a magnet and the outermost electron as a metal ball. The stronger the magnet pulls, the harder it is to remove the ball. Therefore, more energy is needed.

Trend in the Periodic Table

➡ Across a Period

Ionization energy increases from left to right.

➡ Higher

⬇ Down a Group

Ionization energy decreases from top to bottom.

⬇ Lower

Why Does This Happen?

  • Across a period, atomic radius decreases.
  • The nucleus attracts the outer electrons more strongly.
  • More energy is required to remove an electron.

  • Down a group, more electron shells are added.
  • The outermost electron is farther from the nucleus.
  • It is easier to remove the electron.
🧠 Memory Trick

Smaller Atom → Stronger Attraction → Higher Ionization Energy

Exam Tip

Atomic radius and ionization energy are generally opposite trends.

Property Across a Period Down a Group
Atomic Radius ⬇ Decreases ⬆ Increases
Ionization Energy ⬆ Increases ⬇ Decreases

Periodic Properties

Electron Affinity

📖 Definition

Electron affinity is the amount of energy released when a neutral gaseous atom gains an electron to form a negative ion (anion).

💡 Easy Explanation

Imagine atoms are trying to catch an extra ball (electron). Some atoms grab the ball very eagerly, while others are less interested. The stronger an atom attracts an extra electron, the higher its electron affinity.

Periodic Trend

➡ Across a Period

Electron affinity generally increases from left to right.

➡ More Energy Released

⬇ Down a Group

Electron affinity generally decreases from top to bottom.

⬇ Less Energy Released

Why Does This Happen?

  • Across a period, atoms become smaller.
  • The nucleus attracts an incoming electron more strongly.
  • Therefore, more energy is released.

  • Down a group, atomic size increases.
  • The incoming electron is farther from the nucleus.
  • So less energy is released.
🧠 Memory Trick

Smaller atoms grab extra electrons more strongly.

Small Atom → Strong Attraction → High Electron Affinity

Examples

Element Electron Affinity
Chlorine (Cl) Very High
Fluorine (F) Very High
Neon (Ne) Almost Zero

Important Notes

  • ⭐ Halogens have very high electron affinity.
  • ⭐ Noble gases have almost no electron affinity because their outer shell is already complete.
  • ⭐ Electron affinity helps explain why non-metals easily gain electrons.

Periodic Properties

Electronegativity

📖 Definition

Electronegativity is the tendency of an atom to attract the shared electrons of a chemical bond towards itself.

💡 Easy Explanation

Imagine two children pulling a rope. The child who pulls the rope harder is like the atom with higher electronegativity—it attracts the shared electrons more strongly.

Trend in the Periodic Table

➡ Across a Period

Electronegativity increases from left to right.

➡ Higher

⬇ Down a Group

Electronegativity decreases from top to bottom.

⬇ Lower

Why Does This Happen?

  • Across a period, atoms become smaller.
  • The nucleus attracts bonding electrons more strongly.
  • Therefore, electronegativity increases.

  • Down a group, atomic size increases.
  • The bonding electrons are farther from the nucleus.
  • So electronegativity decreases.
🧠 Memory Trick

The smaller the atom, the stronger it pulls shared electrons.

Important Facts

  • ⭐ Fluorine (F) has the highest electronegativity.
  • ⭐ Noble gases usually have little or no electronegativity because they rarely form bonds.
  • ⭐ Metals generally have low electronegativity, while non-metals have high electronegativity.

Periodic Properties

Metallic & Non-metallic Character

🟦 Metallic Character

Metallic character is the tendency of an atom to lose electrons and form positive ions (cations).

Trend

  • ➡ Decreases from left to right across a period.
  • ⬇ Increases from top to bottom in a group.
💡 Large atoms lose electrons more easily, so they are more metallic.

🟩 Non-metallic Character

Non-metallic character is the tendency of an atom to gain electrons and form negative ions (anions).

Trend

  • ➡ Increases from left to right across a period.
  • ⬇ Decreases from top to bottom in a group.
💡 Small atoms attract electrons more strongly, making them more non-metallic.

Why Do These Trends Occur?

As you move across a period, atoms become smaller and hold their electrons more tightly. This makes it difficult to lose electrons but easier to gain them. Therefore, metallic character decreases while non-metallic character increases.


Down a group, atoms become larger because new electron shells are added. The outermost electrons are farther from the nucleus, making them easier to lose. As a result, metallic character increases while non-metallic character decreases.

Examples

Property Example
Most Metallic Element Francium (Fr)
Most Non-metallic Element Fluorine (F)
🧠 Memory Trick

Metals like to Lose electrons.
Non-metals like to Gain electrons.

Lose → Metal
Gain → Non-metal

Important Families

Special Groups of the Periodic Table

Elements in the same group have similar electronic configurations and therefore show similar chemical properties.
Group 1

🟠 Alkali Metals

  • Have 1 valence electron.
  • Very reactive.
  • Soft metals.
  • React vigorously with water.
  • Examples: Li, Na, K.
Group 2

🟡 Alkaline Earth Metals

  • Have 2 valence electrons.
  • Reactive but less than Group 1.
  • Harder and denser than alkali metals.
  • Examples: Be, Mg, Ca.
Groups 3–12

🔵 Transition Metals

  • Good conductors of heat and electricity.
  • Strong and hard.
  • Can form coloured compounds.
  • Many are used as catalysts.
  • Examples: Fe, Cu, Ag.
Group 17

🟢 Halogens

  • Have 7 valence electrons.
  • Highly reactive non-metals.
  • Usually form salts with metals.
  • Examples: F, Cl, Br, I.
Group 18

🟣 Noble Gases

  • Complete outermost shell.
  • Very stable.
  • Almost chemically inert.
  • Examples: He, Ne, Ar.
f-Block

⚪ Lanthanides & Actinides

  • Located below the main table.
  • Known as inner transition elements.
  • Lanthanides are mostly rare earth metals.
  • Most actinides are radioactive.

Chemical Behaviour

Reactions of Elements in the Same Group

Elements belonging to the same group usually have the same number of valence electrons. Because of this, they show very similar chemical reactions.

🟢 Halogens (Group 17)

Halogens react with hydrogen to produce hydrogen halides. 

H₂ + F₂  →  2HF

H₂ + Cl₂ → 2HCl

H₂ + Br₂ → 2HBr

H₂ + I₂  → 2HI

When these gases dissolve in water they produce acids. 

HF  + H₂O → Hydrofluoric Acid

HCl + H₂O → Hydrochloric Acid

HBr + H₂O → Hydrobromic Acid

HI  + H₂O → Hydroiodic Acid

⚗ Reaction with Carbonates

These acids react with carbonate compounds to produce carbon dioxide gas. 

CaCO₃ + 2HCl

↓

CaCl₂ + H₂O + CO₂ ↑
💡 Carbon dioxide gas is released.

📊 Comparing Similar Reactions

Element Reaction Pattern
Fluorine Forms HF
Chlorine Forms HCl
Bromine Forms HBr
Iodine Forms HI

Why are the Reactions Similar?

Every halogen has 7 valence electrons. They all need just one more electron to complete their outer shell.


Therefore, all halogens react with hydrogen in nearly the same way, producing hydrogen halides.

🧠 Remember

Same Group → Same Valence Electrons → Similar Chemical Behaviour

Applications

Importance of the Periodic Table

The periodic table is more than just a chart of elements. It helps scientists, students, engineers, and researchers understand the behavior of matter.
🔍

Predict Properties

Elements in the same group have similar properties. This helps scientists predict the behavior of newly discovered or less familiar elements.

⚗️

Understand Chemical Reactions

The position of an element helps explain how easily it reacts and what compounds it is likely to form.

🧪

Study Trends

Trends such as atomic radius, ionization energy and electronegativity become easy to understand using the periodic table.

🌍

Everyday Uses

From medicines and fertilizers to electronics and construction, knowledge of the periodic table is essential in daily life.

🚀

Scientific Research

Scientists use the periodic table to discover new elements and develop new materials with useful properties.

📚

Learning Chemistry

The periodic table organizes chemistry into a logical system, making it much easier to study and remember.
